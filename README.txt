# nci-gene-trap 


Dependent software:
  - elastix: http://elastix.isi.uu.nl/
  - trackpy: https://github.com/soft-matter/trackpy 
  - FIJI: https://fiji.sc/


Setup:
  - Define the environment variable GENE_TRAP_ROOT to point the git repository.
  - Add GENE_TRAP_ROOT to the PATH.
  - Add GENE_TRAP_ROOT/src/ to the PYTHONPATH.
  - Define the environment variable ELASTIX_PATH (where elastix bin is created) and add it to PATH.
